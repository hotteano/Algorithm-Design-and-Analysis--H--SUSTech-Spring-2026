# Lab8B

## 问题描述

有 $n$ 个男生和 $m$ 个女生，排成一列。要求**任意连续子段**中男女数量之差的绝对值不超过 $k$，求排列总数，对 $12345678$ 取模。

## 核心转化：前缀和

把男生看成 $+1$，女生看成 $-1$。设前缀和为 $S_0=0, S_1, \dots, S_{n+m}$。

任意子段 $[l,r]$ 的男女差 $= |S_r - S_{l-1}|$。

**关键结论**：所有子段合法 $\iff$ 整个序列前缀和的最大值与最小值之差 $\le k$。

即：只要控制整条前缀和曲线的**峰谷差**即可。

## 朴素 DP

如果只记录 $(i, j)$，无法判断"在末尾加一个男生/女生后是否合法"——因为不知道之前的前缀和分布。

因此最朴素的正确状态需要记录历史最值：

$$
dp[i][j][mn][mx]
$$

表示用了 $i$ 男、$j$ 女，历史前缀和最小值为 $mn$，最大值为 $mx$ 的方案数。

当前前缀和 $cur = i - j$（固定值）。转移时，加男生 $\to cur' = cur + 1$，更新 $mn' = \min(mn, cur')$，$mx' = \max(mx, cur')$，检查 $mx' - mn' \le k$。

$mn, mx$ 的范围是 $[-(n+m), n+m]$，状态数为 $O(nm(n+m)^2)$，太大。

## 状态压缩：$d$ 与 $r$

由于 $cur$ 已由 $(i,j)$ 确定，我们不需要直接存 $mn$ 和 $mx$，而是存两个相对量：

- $d = cur - mn$：当前点离历史谷底多远（$0 \le d \le r$）
- $r = mx - mn$：峰谷差，即历史波动范围（$r \le k$）

状态变为 $O(k^2)$：

$$
dp[i][j][d][r]
$$

## 转移方程

当前状态 $(d, r)$，$cur = mn + d$，$mx = mn + r$。

**放男生**（$cur \to cur + 1$）：

| 条件 | 新高度是否突破原峰顶 | 新状态 |
| ------ | ------------------- | -------- |
| $d < r$ | 否，仍低于峰顶 | $(d+1,\ r)$ |
| $d = r$ | 是，突破峰顶 | $(d+1,\ r+1)$ |

**放女生**（$cur \to cur - 1$）：

| 条件 | 新高度是否跌破原谷底 | 新状态 |
| ------ | ------------------- | -------- |
| $d > 0$ | 否，仍高于谷底 | $(d-1,\ r)$ |
| $d = 0$ | 是，跌破谷底 | $(0,\ r+1)$ |

**注意**：$r+1$ 时若 $r+1 > k$ 则不合法。

## 滚动优化

$i$ 维度只需保留两层（$i-1$ 和 $i$），因为放男生只依赖上一层，放女生在同一层从左到右递推即可。

空间复杂度降为 $O(m \cdot k^2)$。

## 代码框架

```cpp
dp[0][0][0] = 1;  // cur=0, 只用到两层

for (int i = 0; i <= n; i++) {
    swap(cur, pre);
    清空 cur 层;
    for (int j = 0; j <= m; j++) {
        // 从 (i-1, j) 放男生
        if (i > 0)
            for (d, r) if (dp[pre][j][d][r])
                if (d < r)    dp[cur][j][d+1][r] += val;
                else if (d == r && r < k)
                              dp[cur][j][d+1][r+1] += val;
        // 从 (i, j-1) 放女生
        if (j > 0)
            for (d, r) if (dp[cur][j-1][d][r])
                if (d > 0)    dp[cur][j][d-1][r] += val;
                else if (d == 0 && r < k)
                              dp[cur][j][0][r+1] += val;
    }
}

ans = sum(dp[cur][m][d][r]) for all d,r;
```
