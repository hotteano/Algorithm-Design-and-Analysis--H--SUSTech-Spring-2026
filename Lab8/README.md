# 习题详解

## LAB8A

本题是一个经典的倍增习题，我们利用任何组中的选择可以被二进制表示来压缩表示，我们避免将每一组$q[i]$全部展开导致$O(qnW)$的复杂度。通过倍增，我们的复杂度为$O(\log q n W)$。

对于每一个组$i$，我们将其拆分为$1+2+4+...+2^{k}+r$，其中$r$为幂方和之外的数。我们注意到，任何在$[0,q]$之间的数都可以被分成的$k+1$组的物品来表示。这是显然的，首先，我们可以表示$[0, 1 + 2 + \ldots 2^{k}]$中的所有数，如果我们选$r$，我们就可以表示$[r, 1 + 2 + \ldots 2^{k}+r]$的所有数，从而我们对于这$k+1$物品的选取就可以表示对于同种物品任何数量的选取。证毕。
