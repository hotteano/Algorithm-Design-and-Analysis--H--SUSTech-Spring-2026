   • Def. An augmenFng path P is a simple s-t path in the residual network Gf.
   • Def. The boIleneck capacity of an augmenFng path P in Gf , denoted by
     boIleneck(Gf , P), is the minimum residual capacity of edges in P.
   • Key property. Consider a ﬂow network G = (V, E, s, t, c) with ﬂow f and
     augmenFng path P in Gf . ALer calling the following Augment( f, c, P), the
     returned f ʹ is a ﬂow in G and v(f ʹ) = v(f) + boIleneck(Gf , P).

                                                  δ = bottleneck(Gf, P)
                                                  foreach e Î P {
                                                     if (e Î E) f(e) = f(e) + δ

   • Ford-Fulkerson (FF) algorithm.
             Ø        Start with f(e) = 0 for each edge e ∈ E.
             Ø        Find an s-t path P in the residual network Gf .
             Ø        Augment ﬂow along path P.
             Ø        Repeat unIl you get stuck.
                         Ford-Fulkerson(G, s, t, c) {
                            foreach e Î E: f(e) = 0
                            Gf = residual network of G with respect to flow f

                                 while (there exists an augmenting path P in Gf) {
                                    f = Augment(f, c, P)
                                    update Gf
                                 }
                                 return f

               Ford-Fulkerson Algorithm: Analysis
   • Assumption. Every edge capacity ce is an integer between 1 and C.
   • Integrality Invariant. Throughout FF, every edge flow f(e) and residual
     capacity cf (e) are integers.

   • Theorem. FF terminates after at most v(f*) ≤ nC augmenting paths,
     where f* is a max flow. (Assume flow network has no parallel edges.)
   • Pf. Consider cut where A = { s } and note that each augmentation
     increases the value of the flow by at least 1. ▪
   • Corollary. The running time of Ford-Fulkerson is O(mnC).
   • Pf. Can use either BFS or DFS to find an augmenting path in O(m) time. ▪
